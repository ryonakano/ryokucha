ellipsize

Description:

public EllipsizeMode ellipsize { get; set; }

The preferred place to ellipsize the string, if the label does not have enough room to display the entire string.

If this property is set to Pango.EllipsizeMode.NONE, no tooltip text is se for each row. Otherwise, tooltip text is set for each row from its label text.
